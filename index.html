The Web Console
Logs information associated with a web page: network requests, JavaScript, CSS,
security errors and warnings as well as error, warning and informational messages expl icitly logged by
JavaScript code running in the page context.
1. To open the console screen in a webpage, right click on a blank side of the webpage.
2. Then click on inspect.
3. Click on console.
4. Here you can see if any errors are there and there are many other sub divisions on the inspect options.
